Run terminal in platformio (icon next to serial monitor on lower bar)
Enter command>

2. Use ESP32 download tool to flash.
Select ESP32 DownloadTool
Configure as follows:

0x1000 bootloader_dio_40m.bin 
0x8000 partitions.bin 
0xe000 boot_app0.bin 
0x10000 firmware.bin

3. if there are issues, try different UART converter (ESP-prog instead built in on board)
and update esptool folder from
https://github.com/espressif/esptool

4. In case of issue:
"A fatal error occurred: Failed to connect to ESP32: Timed out waiting for packet header"
Several solutions:
* short GPIO0 to GND in connecting (sync) phase. Short is easiest on U$2 connector, pins 4+6 (ones closest to R7)
* Or change C16 from 100n to 4.7uF (or 10uF).
* or use some different version of esptool? (not tested)
